# [BOJ 12100 - 2048 (Easy)](https://www.acmicpc.net/problem/12100)


### 문제
2048의 기본 이동 조건을 만족하는 조건 아래, 5번 이동이 가능하다고 할 때 나올 수 있는 최대 타일의 값을 출력 

### 입력
첫째 줄에 보드의 크기 `N`

둘째 줄부터 `N`개의 줄에 게임판의 초기 상태(2차원 배열)

### 풀이
재귀적으로 네 방향에 대해 탐색하고, 5라운드에 도달했을 때 현재 게임 맵에서의 최대 값을 반환한다. 그리고 탐색한 네 방향 가운데 반환값이 가장 큰 값을 최댓값으로 저장하며 최대값을 첫 재귀 호출문까지 반환한다. 네 방향으로 최대 5번까지 이동할 수 있으므로 방향별 탐색 횟수는 `4^5`로 `1024`가 나오며, 각 이동마다 최대 `20 x 20`이 크기를 탐색하기 때문에 모든 경우를 탐색하더라도 시간초과가 발생하지 않는다.
